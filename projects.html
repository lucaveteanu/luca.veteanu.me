Game Development

Knight Adventure

A turn-based RPG adventure game built entirely in JavaScript, running on HTML5 Canvas. Players embark on an epic quest, battling monsters, collecting treasures, and upgrading their knight's abilities. The game features custom sprite animations, strategic combat mechanics, an inventory system, and progressive difficulty levels.

This project challenged me to implement game state management, collision detection, and create an engaging user experience entirely in the browser. It combines my love for gaming with my passion for coding, and serves as a practical example in my teaching curriculum.
